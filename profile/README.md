# Osintly 📡
> One workspace for every OSINT investigation.

## What is Osintly?

[Osintly](https://osint.ly) is a **unified OSINT platform**. Instead of juggling a dozen separate tools, you search across pseudonyms, emails, phone numbers, IPs, domains, and crypto addresses from a **single workspace** — and get enriched, correlated results in real time.

Built for investigators, researchers, and security professionals who need reliable intelligence, fast.

---

## 🛠️ Search Capabilities

Osintly supports six target types in a single unified search:

| Target | Description |
|---|---|
| 🧑‍💻 **Pseudonym** | Investigate usernames across multiple platforms |
| ✉️ **Email** | Uncover email usage across services and detect breaches |
| 🌍 **IP Address** | Geolocation, metadata, and infrastructure analysis |
| 📞 **Phone Number** | Retrieve information linked to phone numbers |
| 🕵️ **Domain** | Ownership, hosting details, and domain-related data |
| 💰 **Crypto Address** | Track transactions and analyze wallet activity |

---

## ✨ Platform Features

- **🔍 OSINT Search** — One input, six target types, dozens of data sources, streamed in real time.
- **🤖 AI Analyst** — Chat over results, generate timelines, filter by signal, and search the web for context.
- **🚨 Monitoring & Alerts** — Set monitors on any query and get notified via email, Discord, or Telegram.
- **📁 Investigation Projects** — Organize searches, files, and notes into cases, solo or with a team.
- **👥 Team Collaboration** — Invite members, assign roles, and share credits.
- **💧 Leaks & Breaches** — Surface compromised data directly within your search results.
- **📤 Export & Sharing** — Export findings as PDF, JSON, or DOCX depending on your plan.

## 📂 About This Repository

This repository is dedicated to the **development and publication of SDKs** for the Osintly API. Our goal is to provide developers with the tools needed to integrate Osintly's OSINT capabilities directly into their applications and workflows.

> ⚠️ The API is documented but **not yet available for purchase**.

### API Overview

The Osintly REST API supports:
- **Search** — Create searches and stream real-time execution updates via SSE
- **Leaks** — Retrieve breach datasets linked to a search
- **Usage** — Track credit consumption and limits
- **Webhooks** — Dispatch events to your infrastructure
- **Auth** — Bearer token authentication

**Base URL:** `https://api.osint.ly`
